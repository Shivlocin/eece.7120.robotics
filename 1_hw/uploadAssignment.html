        2. Homework 1 

          Due Wednesday Sept 11, 2019

           

          1. Create a git repo for code from this class. Give it a unique name that you don’t mind sharing. I recommend bitbucket because they give free academic repos. Use your official UML email for the account. Make sure the repo is PRIVATE until the semester is over.
          2. Create appropriate logins for these services using your official UML email:
            1. Duckietown https://www.duckietown.org/site/register
            2. Docker Hub https://hub.docker.com/
          3. Install docker and docker-compose
          4. Install ROS in a container or VM of your choice
          5. Install dts and otherwise complete duckietown laptop install https://docs.duckietown.org/DT19/opmanual_duckiebot/out/laptop_setup.html
          6. Run a demo:
            1. Running the demo: in a terminal run
              1. git clone git://github.com/duckietown/challenge-aido_LF-baseline-duckietown.git
              2. cd challenge-aido1_LF1-baseline-duckietown/local
              3. docker-compose -f docker-compose-lf.yml pull
              4. docker-compose -f docker-compose-lf.yml build
              5. docker-compose -f docker-compose-lf.yml up
              6. The demo should now be running (it may take awhile to start)
            2. Connecting to the demo:
              1. In another terminal, run: dts start_gui_tools lanefollow --network=local_gym-duckietown-net –sim
              2. If that command fails, check which networks docker is using with “docker network ls” and change the network parameter to whichever one includes the string “gym-duckietown”
              3. Poke around using rostopic and rosnode. Record the output of “rostopic list” to turn in.
              4. Try to use rqt tools like rqt_graph. Note that you may need to work out x-forwarding so this is optional for now.
            3. If you run into troubles:
              1. Work with your classmates
              2. Talk to me
              3. Worst case, turn in a screenshot of the progress you made and we will fix it in class next week
            4. Reference for further information: https://docs.duckietown.org/DT19/AIDO/out/ros_baseline.html
          7. Turn in to blackboard a PDF with:
            1. Your name
            2. Link to your git repo
            3. Username on Bitbucket or whichever service you use
            4. Duckietown username
            5. Docker Hub username
            6. The output of “rostopic echo” run in 6b.
          8. Review Duckiebot Operation Manual (https://docs.duckietown.org/DT19/opmanual_duckiebot/out/) and be prepared to build a Duckiebot next class!

           

          Grading: Each of the items 7b-7f are worth 20% of this homework grade. Unlike other assignments, late work WILL be accepted for up to two weeks after the due date for full credit.
